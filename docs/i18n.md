# i18n Kılavuzu

## Genel Bakış

beni-iceri-al-DPI özel bir i18n sistemi kullanır (`src/i18n.js`). Dış kütüphane yerine basit bir nesne yapısı tercih edilmiştir; bu Tauri bundle boyutunu küçültür ve tam kontrol sağlar.

**Desteklenen Diller:**
- `tr` — Türkçe (varsayılan)
- `en` — İngilizce

---

## Sistem Yapısı

`i18n.js` aşağıdaki yapıya sahiptir:

```js
const translations = {
  tr: {
    buttonConnect: "BAĞLAN",
    buttonDisconnect: "BAĞLANTIYI KES",
    logPortInUse: (port) => `Port ${port} meşgul, yeni port deneniyor...`,
    // ...
  },
  en: {
    buttonConnect: "CONNECT",
    buttonDisconnect: "DISCONNECT",
    logPortInUse: (port) => `Port ${port} in use, trying next port...`,
    // ...
  }
};

export function t(lang, key, ...args) {
  const value = translations[lang]?.[key] ?? translations["en"][key];
  return typeof value === "function" ? value(...args) : value;
}
```

---

## Kullanım

### Statik Metin

```jsx
import { t } from "./i18n";

// config.language değeri "tr" veya "en"
<button>{t(config.language, "buttonConnect")}</button>
```

### Dinamik Mesaj (Parametreli)

```js
// i18n.js'teki tanım:
logPortInUse: (port) => `Port ${port} meşgul...`,

// Kullanım:
addLog(t(lang, "logPortInUse", currentPort), "warning");
```

### Dil Değişimi

Config state değiştiğinde tüm `t()` çağrıları otomatik güncellenir çünkü React bileşenleri yeniden render olur.

## Yeni Dil Ekleme

### 1. `i18n.js`'e yeni dil nesnesi ekleyin

```js
const translations = {
  tr: { /* ... */ },
  en: { /* ... */ },
  de: {                          // Yeni: Almanca
    buttonConnect: "VERBINDEN",
    buttonDisconnect: "TRENNEN",
    logPortInUse: (port) => `Port ${port} belegt, versuche nächsten...`,
    // Tüm anahtarları ekleyin
  }
};
```

### 2. `constants.js`'e dil seçeneğini ekleyin

```js
export const LANGUAGES = [
  { code: "tr", label: "Türkçe" },
  { code: "en", label: "English" },
  { code: "de", label: "Deutsch" },  // Yeni
];
```

### 3. `Settings.jsx` dil seçicisini güncelleyin

Dil listesi `LANGUAGES` sabitinden okunuyorsa otomatik görünür; aksi halde dropdown seçeneklerine ekleyin.

## Eksik Çeviri Yönetimi

`t()` fonksiyonu istenen dilde anahtar bulamazsa İngilizce'ye düşer:

```js
const value = translations[lang]?.[key] ?? translations["en"][key];
```

Eğer İngilizce'de de yoksa `undefined` döner. Bu durum log mesajında `undefined` görünmesine yol açar; yeni anahtar eklerken her iki dile de eklenmesini sağlayın.

---

## Dil Kalıcılığı

Seçilen dil `beni-iceri-al-DPI_config` localStorage anahtarı içinde `language` alanı olarak saklanır:

```json
{
  "language": "tr",
  "dpiMethod": 1,
  ...
}
```

Uygulama başlangıcında bu değer okunur; tarayıcı dilinden otomatik tahmin yapılmaz.
